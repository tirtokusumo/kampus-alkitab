# CMS Kampus Alkitab — Phase 1 (Fondasi)

Plan ini difokuskan pada **fondasi CMS** yang bisa diselesaikan dalam satu sesi dan **tidak mengganggu website yang sudah berjalan**. Semua perubahan bersifat **additive** — hanya menambah file dan route baru, tanpa mengubah halaman publik yang sudah ada.

## Prinsip Keamanan

> [!TIP]
> **Zero-breaking-change approach:**
> - Semua file CMS baru, tidak ada file lama yang dihapus
> - Landing Page & halaman publik lainnya **tidak diubah** di Phase 1
> - Admin panel adalah route terpisah (`/admin`) yang tidak mempengaruhi UX publik
> - Phase berikutnya (menghubungkan CMS ke halaman publik) bisa dilakukan kapan saja

## Scope Phase 1

| Yang dikerjakan ✅ | Yang ditunda ⏳ |
|---|---|
| Admin Layout + Dashboard | Membuat Landing Page dinamis |
| Page Manager (CRUD) | Block Editor visual (drag-drop) |
| Menu Builder (header & footer) | Dynamic rendering di halaman publik |
| Theme/Color Settings | Supabase integration |
| Site Settings (logo, nama, tagline) | Media Manager (upload gambar) |
| CMS Data Layer (localStorage) | Multi-page block editing |
| Admin Route Protection | — |

## Proposed Changes

### 1. CMS Data Layer

#### [NEW] `src/cms/cmsData.js`
- Default data: site settings, menus, pages list, theme colors
- CRUD helpers: `getCmsData()`, `saveCmsData()`, `updateSiteSettings()`, `updateMenus()`, `updateTheme()`, `addPage()`, `updatePage()`, `deletePage()`
- Data disimpan di localStorage, load otomatis saat app start

#### [NEW] `src/cms/CmsContext.jsx`
- React Context + Provider untuk data CMS
- Auto-load dari localStorage
- Menyediakan semua fungsi update ke admin components

## Verification Plan

### Browser Testing
1. Login normal (bukan admin) → akses `/admin` → harus redirect ke `/`
2. Login sebagai `admin@kampusalkitab.com` → akses `/admin` → berhasil masuk
3. Navigasi semua section CMS (Pages, Menu, Theme, Settings)
4. Ubah warna primary → verifikasi CSS berubah di halaman publik
5. Tambah page → verifikasi muncul di tabel
6. Edit menu → verifikasi data tersimpan di localStorage
